# Gravitron
---
### About
Gravitron is a small project simulating the Solar System.
---
### Controls
RMB | Select/Deselect Planet \
RMB | Move Camera \
WheelUp |  Zoom In \
WheelDown | Zoom Out \
\+ | Zoom In \
\- | Zoom Out \
Ctrl + + | Speed Simulation up \
Ctrl + - | Speed Simulation up \
Shift | Deselect Planet
